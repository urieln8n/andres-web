# Arquitectura de la mega web personal

## Capas

1. **Capa pública SEO**
   - Home
   - Servicios
   - Nichos
   - Local Barcelona
   - Blog
   - Proyectos

2. **Capa de conversión**
   - Auditoría gratis
   - Formularios
   - CTA WhatsApp

3. **Capa CRM**
   - API `/api/leads`
   - Supabase table `leads`
   - Dashboard visual `/dashboard`

4. **Capa automatización**
   - Webhook `LEAD_WEBHOOK_URL`
   - n8n/Make/Zapier
   - Email/WhatsApp follow-up

La web no es un portfolio. Es un sistema comercial.
