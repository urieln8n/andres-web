# Checklist de produccion

- `NEXT_PUBLIC_SITE_URL=https://andresrendon.com`
- `NEXT_PUBLIC_SUPABASE_URL` configurada.
- `NEXT_PUBLIC_SUPABASE_ANON_KEY` configurada.
- `SUPABASE_SERVICE_ROLE_KEY` solo en servidor si se usa.
- Dominio `andresrendon.com` conectado en Vercel.
- `npm run lint` sin errores.
- `npm run typecheck` sin errores.
- `npm run build` sin errores.
- Sitemap y robots accesibles.
- Formulario probado con Supabase o webhook.
- Dashboard protegido antes de usar datos reales.
